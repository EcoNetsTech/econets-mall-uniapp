## 🐯 商城简介

> 有任何问题，或者想要的功能，可以在 Issues 中提出。
>
> 😜 给项目点点 Star 吧，这对我们真的很重要！

* 基于 uni-app + Vue3 开发，支持微信小程序、微信公众号、H5 移动端，未来会支持支付宝小程序、抖音小程序等
* 支持 SaaS 多租户，可满足商品、订单、支付、会员、优惠券、秒杀、拼团、砍价、分销、积分等多种经营需求
